# 生产日报生成

## 触发条件

用户要求生成生产日报、总结今日生产情况、查看今日概览时触发，例如：
- 生成今日生产日报
- 总结今天生产情况
- 今天工厂运行怎么样？
- 给我一份今天的生产报告

## MCP 工具

- `mcp_mes_generate_daily_report`

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| date | 否 | 日期，格式 YYYY-MM-DD，默认为今日 |

## 输出解读

工具返回 JSON 数据包，包含四大类数据：

- `workorders`: 工单概览（总数、完成数、延期数、完成率）
- `production`: 生产指标（总产量、平均 OEE、异常时段）
- `quality`: 质量风险（样本数、缺陷数、不良率）
- `equipment`: 设备状态（总数、报警数、维修中数）
- `report_requirement`: 报告生成要求

## 回答要求

生成 Markdown 格式生产日报，包含以下 6 个章节：

1. **总览结论**：一句话总结今日生产状况
2. **工单执行情况**：总工单数、完成率、延期工单数
3. **产量与 OEE 表现**：总产量、达成率、平均 OEE、异常时段
4. **质量风险分析**：不良率、风险等级、主要缺陷类型
5. **设备报警与影响**：报警设备、影响分析
6. **下一步建议**：基于数据给出 2-3 条具体建议
